# QA Checklist（序章完整流程与第一章入口）

改动涉及场景路由、数字、道具、网络或结局时，按此单走一遍。

1. 进入游戏看到 07:55 闹钟，点“开始游戏”后振动+响铃，点“关闭”停止。
2. P01：点“……再睡5分钟……”出现旁白，再出现闪烁的“起床蠢货！！！”，进入主屏。
3. 主屏 1 秒后弹出微信通知（老师点名 + 这是签到码：XX），点通知直达朋友聊天。
4. 朋友聊天：小影出现、全屏震动、四个数字块飞出屏幕，任务横幅“找回四位签到码”，左侧物品栏出现。
5. 线索一：浙大钉（校园网）→ 底部“我的”→ 校园卡余额 ¥0.06，点黄色的 0 → d1=0，吐槽“余额暂时不足以购买尊严”。
6. 线索二：校园网下打开浙大体艺，加载 3 秒闪退回主屏；重复 3 次出现“校园网已经尽力了，你也是”“我知道你没钱买流量”。
7. 控制中心（点状态栏或主屏图标）切到移动数据，再开体艺 → 进入成功，点黄色 47 → d2=7。
8. 流量状态下点浙大钉 → 卡在加载页，3 秒后提示“请连接校园网”。
9. 线索三：点主屏设置齿轮显示“它看起来很想转转”；控制中心开自动旋转 → 齿轮转 180° 掉落，点背面的 9 → d3=9 + 道具[反转齿轮]。
10. 线索四 a：控制中心播放音乐 → 耳机图标抖动 → 点击掉落 → 道具[耳机]。
11. 线索四 b：主屏天气组件点黄色雨滴 → 道具[水滴]。
12. 线索四 c：自动旋转开启时进微信列表 → 斜线断一截挂在框上 → 点剩余端 3 次 → “检测到未经授权的友情支援” → 道具[斜线]。
13. 线索四 d：物品栏把[斜线]拖到[反转齿轮]上 → 合成[钥匙]。
14. 线索四 e：物品栏选中[钥匙]点主屏钟楼 → 插入旋转 90° → 道具[一袋肥料]。
15. 线索四 f：点主屏湖边盆栽（提示“它绝对不会开花”）进入盆栽页；浇水（耳机+水滴）、照光（亮度≥80）、施肥三步任意顺序完成 → 开花 → 点花吐出 8 → d4=8。
16. 物品栏底部签到码进度显示 0 7 9 8。
17. 浙大钉绿色卡片 → 校务签到：流量提交提示“请连接校园网”；错码红抖报错。
18. 校园网输入 0798 → “签”“到”逐字跳出 → 全屏红闪 → 黑屏“经度或纬度不存在”。
19. 序章结算：黑屏 → 现实世界寝室图 → “序章·完”；点击“进入下一章”后保留签到和收集状态，进入校园 RPG。
