# 第二章图书馆完整流程草案：022 的占座书包

> 文档角色：剧情真源。对白、叙事顺序、任务文案和第三章衔接以此文件为准。

## Summary

本章从玩家获得校园 RPG 操作权、准备去图书馆找“系统的朋友”开始。核心体验改为：**RPG 图书馆地图探索为主，手机 App 负责加工/认证道具，CC98 保留 `bd` 作为公示机制，最终解除 022 号座位的占座书包，联系到系统朋友，并过渡到第三章“寻找借走签到记录的书”。**

默认保留既有设定：体艺“开始课外锻炼”已经让小人可行走；天气不再作为本章核心谜题；CC98 保留 `bd` 和十大；整体风格继续走像素校园、手机页面物理化、荒诞吐槽、轻微 meta。

## 完整流程

### 1. 新界面：RPG 图书馆地图

图书馆不是手机页面，而是校园 RPG 内部地图。像素俯视角，区域包括：

```text
入口门禁
前台
失物招领
自助借还机
馆藏检索终端
打印机
二楼南区书架
二楼南区座位区
022 号座位
```

视觉设计：

```text
地砖偏灰蓝，书架是深棕像素块，座位区用小桌椅格子表示。
可交互物体上方出现小白点或轻微闪烁。
手机外层仍保留，表示这是手机里的校园 RPG。
```

玩家进入后，系统提示：

```text
系统：她以前在二楼南区 022。
玩家：以前？
系统：图书馆的“以前”通常指三分钟前，也可能指三年前。
旁白：这取决于系统想不想负责。
```

玩家需要先点击入口闸机旁的小屏查看入馆记录。小屏展开后显示：

```text
07:55：进入主馆
08:02：到达二楼南区 022
二层南区 022：存在未闭合会话
最后活动：未知
```

小屏保留 `08:02 − 07:55` 的核对式，不显示计算结果。玩家点击“记下记录”后，任务才更新。

此时任务从“读取入馆记录”更新为“前往二层南区寻找 022”。让玩家先建立“座位仍有信号”的目标，再进入占座调查。

玩家走到 `022`，发现座位上有一个像素书包。书包旁有纸条。

靠近书包时，座位下方短暂出现三圈信号波纹和 `022 LINK LOST`，随后被书包遮住。演出持续约 1 秒，只用于确认系统朋友确实位于此处。

点击书包：

```text
书包：……
旁白：它沉默得很有产权意识。
玩家：一个书包。
系统：别直呼它的真名。
玩家：占座书包？
系统：你非要把 BOSS 名字念出来吗？
```

点击纸条，获得道具：

```text
道具：占座纸条
内容：本人离开三分钟，精神仍在座位上。临时离座规则详见 CC98。
```

任务更新：

```text
调查 022 的占座规则
```

### 8. 浙大体艺：本人来过证明

体艺不再负责“让小人走起来”，而是读取玩家 RPG 地图中已发生的路径。

体艺页面标题：
本人来过证明补录单
页面文案：
检测到室内异常锻炼路线：
寝室 → 图书馆入口 → 022 → 前台 → 书架背面 → 022

状态：待补录
原因：路线真实，但字段不像真的。
三个字段：
到座耗时：__ 分钟
公示编号：__ 号
证明数量：__ 项

答案来源：
7：图书馆入口记录显示 07:55 入馆，08:02 到达 022。
47：CC98 调查帖第 23 楼的楼主编辑写明，旧申请统一使用公示编号 47。
3：旧版规则明确要求三项证明。

错误提示：
第一次：按来源 01/02/03 逐项处理：计算时间差、读取公示编号、数规则条目。
第二次：来源 01 看两次时间，来源 02 看 23 楼楼主编辑，来源 03 看旧规则正文。
第三次及以后：仍有字段与来源不一致。注意：23 是回复楼层，表单需要读取原文中的公示编号。

成功文案：
体艺：补录成功。
体艺：系统已承认你确实来过图书馆，且来得很复杂。
系统：现实终于被填进了表格。
玩家：所以我现在存在了？
系统：在体艺里存在。别要求太高。
获得道具：
本人来过证明
说明：一张证明你来过的证明。它没有证明你为什么要来。


### 9. CC98：提交证据并完成 BD 四位口令

玩家回 CC98 原帖。楼主编辑区出现“上传证据”槽位。

需要拖入四个道具：

```text
旧版离座规则
书包非本人证明
022 座位小票
本人来过证明
```

全部上传后，帖子状态变为：

```text
证据完整度：4/4
当前排名：04
提示：按证据顺序完成 BD 四位热度口令
```

系统先解释：`bd` 是论坛用语“帮顶”。在数字回复上点击 `bd`，该回复的数字会按选择顺序写入口令。图书馆 App 只读取进入十大帖子的四位热度口令。

口令顺序沿用上方四项上传材料：

```text
1. 旧版离座规则：证明要求数量
2. 书包非本人证明：身份通过项数量
3. 022 座位小票：座位号末位
4. 本人来过证明：到座耗时
```

证据上传后新增 `24–31` 楼八条数字回复，数字为 `4 / 3 / 1 / 0 / 5 / 2 / 6 / 7`。玩家需要阅读回复来源，对四条对应回复依次点击 `bd`，组成：

```text
3 0 2 7
```

页面持续显示四个口令槽，支持撤回一位和清空。四位填满后由玩家点击“提交口令”。

错误提示分三层：

```text
1. 提醒顺序跟随证据上传栏
2. 说明四项操作分别是计数、计数、取末位、取耗时
3. 指出公示编号、目标排名、索书号和回复总数属于干扰信息
```

正确提交后：

```text
当前排名：04 → 01
帖子进入十大第 1
标题变红：【十大】022 的书包已经三天没眨眼了
```

旁白：

```text
旁白：问题终于变得足够公开，于是它暂时变得可以处理。
```

任务更新：

```text
前往图书馆 App 提交恢复申请
```

- 玩家能从校园 RPG 进入图书馆，并在 022 触发占座书包剧情。

- 章节结尾必须解锁第三章任务“找到那本借走签到记录的书”。
