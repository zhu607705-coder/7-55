> 核对日期：2026-07-13  
> 核对范围：当前 `src/` 源码、章节开发文档、开发者断点和单文件运行入口。

## 0. 先看结论

当前源码包含三段可调试内容：

1. 第一章：找回签到码 `0798`，完成签到，并抓住准备离场的旁白圆圈。
2. 第二章：使用第一章取得的校园卡和物品栏恢复移动能力，进入 RPG 图书馆，处理 `022` 占座书包。
3. 第三章：`求是潮 755` 三车道骑行小游戏。

当前正式流程在完成 `022` 对话后会同一次写入 `friend_contacted`、第三章任务和 `bikeArcade.unlocked = true`。旧存档若已经完成对话但仍处于锁定状态，载入时会自动迁移修复。

旧版第二章的 `63 楼调查帖 → 18 条 ac01 → 47/12/1 引用链 → 路线截图 → temporary PASS` 已退出当前正式流程，不要再按这条旧攻略调试。

## 1. 启动、存档与公共操作

### 1.1 启动

- 开发版：打开 `http://127.0.0.1:5173/`。
- 单文件版：直接打开 `demo/index.html`。
- 第一次进入时从 `07:55` 闹钟页面开始。

### 1.2 手机公共操作

- **打开控制中心**：点击顶部状态栏右侧的 `ZJUWLAN / 流量 / 17% / 电池` 区域。
- **打开物品栏**：获得至少一件道具后，点击手机左侧的背包图标。
- **移动物品栏位置**：拖动物品栏空白处或把手，物品栏只会上下移动并吸附在左侧。
- **查看道具说明**：单击道具。
- **合成道具**：把一个道具拖到另一个道具上。
- **使用道具**：把道具拖到场景中的目标位置，松开鼠标或手指。
- **立即存档**：控制中心 → `游戏进度` → `立即保存`。
- **重置剧情**：控制中心 → `游戏进度` → `重置剧情进度` → `确认重置`。重置会清除章节、道具和谜题状态，保留玩家编辑过的 CC98 帖子。

### 1.3 RPG 公共操作

- 键盘移动：方向键或 `WASD`。
- 交互：靠近目标后按交互键；触屏使用右下角方向键和 `A`。
- 打开手机：横屏 RPG 右上角 `打开手机`。
- 全屏：右上角 `全屏`，或按 `F`。
- 图书馆道具：从横屏左侧/上方的 RPG 道具栏拖到场景目标，不使用手机竖屏物品栏。

## 2. 第一章：找回签到码 0798

### 2.1 从闹钟进入手机

1. 点击 `开始游戏`，闹钟开始响。
2. 点击 `关闭`。
3. 点击 `……再睡5分钟……`。
4. 出现 `起床蠢货！！！` 后，点击 `进入手机主界面`。
5. 等待微信通知，打开 `微信` → `朋友`。
6. 看完小影把四位签到码打散的演出。

成功判据：任务提示变为 `找回四位签到码`，主界面开始出现可收集机关。

### 2.2 第一位：校园卡余额 0

1. 签到码散落后，点击主页通知 `浙大钉：查看寝室定位`。
2. 在浙大钉点击任务卡 `查看寝室定位`，或点击应用区的 `校园地图`。
3. 进入横屏寝室 RPG，点击右侧书桌上的校园卡，获得 `校园卡`。
4. 点击 RPG 右上角 `打开手机`，回到浙大钉。
5. 点击 `电子校园卡`。
6. 找到余额 `0.06` 中被标黄的第二个 `0`。
7. 点击黄色 `0`。

获得：`校园卡`、物品栏和第 1 位 `0`。

### 2.3 第二位：浙大体艺次数 7

1. 点击顶部状态栏右侧，打开控制中心。
2. 把网络从 `ZJUWLAN` 切换到 `移动数据`。
3. 回到主界面，打开 `浙大体艺`。
4. 等待加载完成，点击高亮的 `47` 中的 `7`。

获得：第 2 位 `7`。

补充：第一章中使用校园网打开体艺会在约 3 秒后闪退。连续失败三次会触发额外旁白，这不是通关条件。

### 2.4 第三位：设置齿轮背面的 9

1. 回到主界面，点击 `设置` 图标，得到“它看起来很想转转”的提示。
2. 打开控制中心，开启 `自动旋转`。
3. 关闭控制中心并等待约 1.5 秒。
4. 设置齿轮会旋转半圈并掉落。
5. 点击掉落后的齿轮背面。

获得：第 3 位 `9`；道具 `反转齿轮`。

### 2.5 第四位前置 A：获得钥匙和肥料

1. 打开微信联系人列表，不进入朋友聊天。
2. 开启控制中心的 `自动旋转`，回到微信联系人列表。
3. 等待朋友头像上的斜线断下一半并挂住。
4. 连续点击朋友头像三次。
5. 获得道具 `斜线`。
6. 在物品栏中把 `斜线` 拖到 `反转齿轮` 上，合成 `钥匙`。
7. 回主界面，把 `钥匙` 拖到钟楼中部的黑色锁孔。
8. 等待插入和旋转演出结束。

获得：道具 `一袋肥料`。

### 2.6 第四位前置 B：获得盛水的耳机

1. 在主界面天气卡片中点击可收集的雨滴，获得 `水滴`。
2. 打开控制中心，点击播放按钮，让 `早八进行曲` 开始播放。
3. 点击正在抖动的耳机图标，等待耳机掉落。
4. 在物品栏中把 `水滴` 和 `耳机` 合成。

获得：道具 `盛水的耳机`。

### 2.7 第四位：让盆栽开花得到 8

1. 点击主界面左下方盆栽，进入盆栽页面。
2. 把 `盛水的耳机` 拖到盆栽上。
3. 把 `一袋肥料` 拖到盆栽上。
4. 打开控制中心，把亮度拉到 `80` 或更高，然后返回盆栽页面并触发光照。
5. 三项条件完成后盆栽开花。
6. 点击花朵吐出的数字。

获得：第 4 位 `8`。完整签到码为 `0798`。

浇水、施肥和光照可以交换顺序。

### 2.8 签到与序章收尾

1. 打开控制中心，把网络切回 `ZJUWLAN`。
2. 主界面打开 `浙大钉` → `学在浙大` → `校务签到`。
3. 输入 `0798`，点击 `签到`。
4. 等待“签”“到”红闪和黑屏演出结束。
5. 黑屏持续约 7 秒后，红色旁白圆圈开始移动。
6. 用鼠标或手指按住圆圈至少 `0.9 秒`，不要提前松开。
7. 看完对话和白屏闪退演出。

成功判据：回到手机主界面，任务进入第二章的 `查看朋友的新消息`。

## 3. 第二章前置：找系统和移动能力

### 3.1 找到系统

1. 等待微信弹窗 `成功了吗`。
2. 打开微信 → 朋友，等待自动回复对话播放完成。
3. 任务变为 `找到系统`。
4. 打开 `浙大钉`。
5. 点击名字旁边由白色虚线圈变成的红色圆圈。
6. 连续点击对话右侧的 `›`。系统确认玩家已经带着校园卡，然后说明需要去图书馆。

成功判据：任务直接变为 `找到移动的办法`。该阶段不会再次要求寻找寝室校园卡。

### 3.2 给角色姓名

1. 在宿舍 RPG 中点击小人，会得到“他听不到你说话”的反馈。
2. 打开手机 → 浙大钉 → `部门黄页`。
3. 输入：
   - 姓名：`林星宇`
   - 学号：`3250100755`
4. 点击拨号/联络按钮。

成功判据：角色上方出现姓名，状态 `characterNamed` 完成。

### 3.3 让角色自动走动

1. 回手机主界面，打开 `浙大体艺`。
2. 点击 `开始课外锻炼`。
3. 点击 `锻炼进行中` 返回宿舍 RPG。

成功判据：小人在宿舍中来回走动，但玩家还不能控制方向。

### 3.4 组装右移箭头

1. 回手机主页，点击最下方第一条推送的三角形头像，获得 `三角形`。
2. 打开天气页面，点击水滴，获得第二章道具 `天气水滴`。
3. 打开微信联系人列表，点击导师头像上的竖线，确认它被黏住。
4. 从物品栏把 `天气水滴` 拖到导师头像。
5. 竖线掉落，获得 `导师竖线`。
6. 在物品栏把 `三角形` 和 `导师竖线` 合成。

获得：可重复使用的 `右移箭头`。

### 3.5 把 0.06 变成 6.00 并购买手柄

1. 打开校园卡余额页。
2. 把 `右移箭头` 拖到余额 `0.06` 上。
3. 小数点右移两位，余额变为 `6.00`。
4. 打开 CC98。
5. 进入二手市场帖子 `6块出游戏手柄，寝室自提`。
6. 点击 `支付 6 元购买手柄`。

获得：`游戏手柄`。宿舍中的自动走动停止，玩家获得方向控制。

### 3.6 完成移动测试并出门

1. 回宿舍 RPG。
2. 用任意方向键或触屏方向按钮移动一次。
3. 任务变为 `可以出门了`。
4. 操作角色走到宿舍出口并交互。

成功判据：进入紫金港校园地图，任务变为 `前往图书馆寻找系统的朋友`。

## 4. 第二章主体：022 的占座书包

### 4.1 进入图书馆并建立三条证据线

1. 在校园 RPG 中前往基础图书馆入口并交互。
2. 进入图书馆后先读取入口处的 `入馆记录`。
3. 前往二楼南区 `022`，检查座位上的书包。
4. 拿起书包旁的 `占座纸条`。
5. 打开手机 → CC98。
6. 把 `占座纸条` 拖到 CC98 搜索框，打开 23 楼剧情帖。

成功判据：任务进入 `证据收集`，需要完成四项证据：旧版规则、书包非本人证明、022 小票、本人来过证明。

帖子中的 5 条 `ac01` 只提供可选内容，不影响通关。

### 4.2 证据一：旧版离座规则

1. 在剧情帖中阅读线索，得到检索词 `三分钟离座法`。
2. 打开浙大钉 → 图书馆 → `馆藏检索`。
3. 输入 `三分钟离座法`，点击 `搜索`。
4. 在五个结果中选择正确书目《三分钟离座法及其例外》；索书号为 `I247.55 / 755`。
5. 获得道具 `索书号 755`。
6. 回横屏图书馆，前往 `文学书架夹层 / 755 号书架`。
7. 从 RPG 道具栏把 `索书号 755` 拖到书架目标。

获得：`旧版离座规则`。

### 4.3 证据二：书包非本人证明

1. 打开手机照片，进入 `IMG_0755.JPG`。
2. 打开控制中心，把亮度降到 `20` 或更低。
3. 返回照片，确认书包标签变得可读。
4. 点击 `生成识别报告`。
5. 获得 `物品识别报告`。
6. 回横屏图书馆，前往失物招领旁的 `物品身份登记机`。
7. 把 `物品识别报告` 拖到机器上。

获得：盖章后的 `书包非本人证明`。

### 4.4 证据三：022 座位小票

1. 回到 `022` 座位。
2. 从 RPG 道具栏把此前保留的 `右移箭头` 拖到桌面夹缝。

获得：`022 座位小票`。右移箭头不会被消耗。

### 4.5 证据四：本人来过证明

这一步需要三份来源先就绪：

- 入馆记录已经读取。
- CC98 调查帖已经建立。
- 旧版离座规则已经取得。

操作：

1. 打开手机 → 浙大体艺。
2. 在 `本人来过证明补录单` 中设置：
   - 到座耗时：`7 分钟`
   - 公示楼层：`47 楼`
   - 证明数量：`3 项`
3. 点击 `提交补录`。

获得：`本人来过证明`。

### 4.6 上传四项证据并把帖子推到十大第一

1. 打开 CC98 → 022 调查帖。
2. 在楼主证据上传区依次上传四项：
   - 旧版离座规则
   - 书包非本人证明
   - 022 座位小票
   - 本人来过证明
3. 四项上传完毕后，处理 A-E 五条回复。
4. 只对有效回复执行 `bd`：`A`、`C`、`E`。
5. `B`、`D` 是混淆项，不会推动排名。

排名变化：`04 → 03 → 02 → 01`。

成功判据：页面显示 `十大第一`，浙大钉图书馆开放 `022 恢复申请`。

### 4.7 生成 PASS 并清退书包

1. 打开浙大钉 → 图书馆 → `022 恢复申请`。
2. 提交三项恢复材料：
   - 书包非本人证明
   - 022 座位小票
   - 本人来过证明
3. 点击生成/签发 PASS。
4. 点击 `回图书馆处理书包`。
5. 在横屏图书馆的 RPG 道具栏中，把 `解除占座 PASS` 拖到 `022` 的书包。
6. 等待书包移动到失物招领。
7. 靠近空出的椅子，点击 `A` 坐下。
8. 继续交互，看完 `022` 对话。

成功判据：任务变为 `找到那本借走签到记录的书`，`libraryFinalsPhase = friend_contacted`。

## 5. 第三章：求是潮 755

### 5.1 进入方式

正式流程完成第二章 `022` 对话后，手机主页会开放 `求是潮`。人工分段调试也可以使用开发者通道：

- `c3-intro`：从 0 米开始。
- `c3-congestion`：从 377 米拥堵段开始。
- `c3-sprint`：从 566 米冲刺段开始。
- `c3-result`：直接查看 755 米完成结算。

### 5.2 玩法

1. 点击 `开始骑行`。
2. 键盘使用 `A / D` 或左右方向键换道；触屏点击底部左右箭头。
3. 躲避自行车、路障和人群。
4. 每次碰撞失去一次机会，总共三次机会。
5. 到达 `755m` 且至少剩余一次机会即通关。

里程节点：

- `188m`：节奏提升。
- `377m`：拥堵升级。
- `566m`：最后冲刺。
- `755m`：完成。

失败后可以 `再骑一次`，最佳距离与尝试次数会保存。

重点查看：

- `currentScene`：当前手机页面。
- `runtimeMode`：`phone` 或 `rpg`。
- `rpgScene` / `rpgCheckpoint`：当前 RPG 地图与位置。
- `ownedItems`：当前拥有道具。
- `digits`：四位签到码收集状态。
- `actOne.phase`：第二章移动前置阶段。
- `ui.libraryFinalsPhase`：图书馆主流程阶段。
- `ui.libraryFinalsPuzzle`：四项证据、bd、PASS、座位恢复的细粒度状态。
- `bikeArcadeChapter`：第三章解锁、尝试次数、最佳距离和完成状态。

### 常见卡点对照

| 现象 | 检查项 | 正确条件 |
| --- | --- | --- |
| 左侧没有背包 | `ownedItems`、`actOne.phase` | 至少有一个道具；旧存档处于 `inventory_required` 时暂时隐藏 |
| 体艺第一章总闪退 | `networkMode` | 切到 `cellular` |
| 角色无法命名 | 姓名与学号 | `林星宇 / 3250100755` |
| 买不了手柄 | `balanceShifted` | 先用右移箭头把 `0.06` 变成 `6.00` |
| 宿舍不能出门 | `manualControlTested` | 买手柄后实际移动一次 |
| 图书馆补录不能提交 | 三份来源 | 入馆记录、CC98 调查帖、旧版规则均已完成 |
| 照片仍反光 | `ui.brightness` | 亮度需 `<= 20` |
| CC98 排名不动 | 上传数和回复 | 先上传 4 项证据，再 bd `A/C/E` |
| 恢复申请没有入口 | `libraryFinalsPhase` | 需要 `top_ten_reached` |
| PASS 拖不上书包 | 运行模式与道具栏 | 回横屏图书馆，用 RPG 道具栏拖拽 |
| 第二章完成后第三章未开放 | 存档版本与 `bikeArcade.unlocked` | 重新载入一次触发 V3 迁移；仍异常时在控制中心重置后复测 |

## 8. 推荐人工回归顺序

每轮代码变更不必从闹钟完整重玩。按以下顺序分段检查：

1. `c1-code-hunt`：四位数字和三组合成配方。
2. `c1-checkin`：签到、7 秒黑屏、按住旁白圆圈。
3. `c1-dorm-card`：第一章宿舍右侧书桌校园卡、物品栏显示和返回手机。
4. `c2-name`、`c2-exercise`、`c2-arrow-assembly`、`c2-gamepad-market`：逐段检查移动谜题。
5. `c2-library-gate`、`c2-seat-arrival`、`c2-occupancy-note`：检查地图入口与 022 到达状态。
6. `c2-catalog`、`c2-photo-report`、`c2-seat-receipt`、`c2-tiyi-proof`：检查四项证据。
7. `c2-cc98-upload`、`c2-bd-rise`、`c2-recovery-form`：检查公示、排名和恢复申请。
8. `c2-pass-apply`、`c2-seat-sit`、`c2-seat-dialogue`、`c2-chapter-exit`：检查清退、坐下、对话和解锁。
9. `c3-intro`、`c3-congestion`、`c3-sprint`：分别检查三个骑行难度阶段。
10. 最后重置存档，从 `c1-alarm` 完整通关一次，确认跨章节状态和存档恢复。
