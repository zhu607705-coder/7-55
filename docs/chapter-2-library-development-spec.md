## 1. 当前版本可沿用与需要替换的开发部分

### 可沿用

1. 手机主框架、状态栏、道具栏、线索条、应用切换。
2. 校园 RPG 的基础移动、方向控制、地图入口。
3. CC98 帖子详情、楼层、回复、bd、排名上升和十大演出。
4. 图书馆 App 的预约/选座视觉基础。
5. 浙大体艺页面基础样式、审核状态、步进器交互。
6. 照片 App 的图片查看与弹层基础。
7. PASS 拖拽到目标区域的交互思路。

### 需要替换

1. 原“63 楼调查帖 + 引用链 + 路线拼接”不再作为第二章主流程。
2. 原“体艺拖动路线 + 自行车碰撞”不再作为本人来过证明的核心玩法。
3. 原“只看楼主推导 7-47-3 恢复口令”改为体艺补录单的明确字段线索。
4. 原“临时 PASS 直接拖回图书馆选座页 022”改为回 RPG 地图，对占座书包使用“解除占座 PASS”。

### 需要新增

1. RPG 图书馆内部地图。
2. 022 占座书包事件。
3. 图书馆馆藏检索页面。
4. RPG 书架背面旧版规则谜题。
5. 失物招领 / 物品身份盖章机。
6. 照片识别书包标签的亮度谜题。
7. 022 座位小票夹缝谜题。
8. 体艺“本人来过证明补录单”。
9. CC98 证据上传 + 八条数字回复中的 BD 四位口令。
10. 图书馆恢复申请页面。
11. 022 座位异常对话与第三章入口。

## 2. 章节定位

本章从玩家已经获得校园 RPG 操作权、准备去图书馆找“系统的朋友”开始。玩家进入图书馆后发现 022 被一个像素书包占用。为了联系 022 中的异常意识，玩家必须证明：

1. 自己本人来过。
2. 座位确实是 022。
3. 当前占用物不具备人格。

通关不是靠读完一个超长论坛帖，而是通过 RPG 地图探索获得实体道具，再用手机 App 处理、认证、公开，最后回到 RPG 地图解除占座。

## 12. 浙大体艺：本人来过证明补录单

当前已开发版中，体艺模块是“路线拖动 + 自行车碰撞 + 7/47/3 审核”。新版改为读取玩家在 RPG 图书馆里已经走过的路径，不再让体艺承担“让小人动起来”的功能，也不再要求玩家靠牵强联想推 7/47/3。

体艺页面标题：

本人来过证明补录单

页面文案：

检测到室内异常锻炼路线：
寝室 -> 图书馆入口 -> 022 -> 前台 -> 书架背面 -> 022

状态：待补录
原因：路线真实，但字段不像真的。

表单顶部直接说明三项材料及取得位置：

1. 图书馆入口小屏 → 到座耗时。
2. CC98 调查帖楼主编辑 → 公示编号。
3. 二楼南区 755 书架后的旧版规则 → 证明数量。

三项材料允许按任意顺序收集，全部取得后才开放提交。界面不得只写“来源 01/02/03”而不解释材料名称和取得位置。

三个字段：

到座耗时：__ 分钟
公示编号：__ 号
证明数量：__ 项

初始值：

到座耗时：5 分钟
公示编号：45 号
证明数量：1 项

正确值：

到座耗时：7
公示编号：47
证明数量：3

答案来源必须明显展示：

1. 7：图书馆入口记录显示 07:55 入馆，08:02 到达 022。
2. 47：CC98 调查帖第 23 楼的楼主编辑写明，旧申请统一使用公示编号 47。
3. 3：旧版规则明确要求三项证明。

新增线索位置：

1. RPG 图书馆入口小屏幕：
   入馆记录：07:55 进入主馆，08:02 到达二楼南区。
   交互：玩家点击小屏展开两条时间；小屏显示 `08:02 − 07:55`，不显示差值。
   结算：首次点击“记下记录”后写入线索状态；完成后仍可点击复查。

2. CC98 第 23 楼楼主编辑：
   原文显示“旧申请统一挂在公示编号 47”。
   `23` 表示回复楼层，`47` 表示审核表需要填写的公示编号。

3. 旧版离座规则：
   恢复 022 需提交三项证明。

错误提示：

第一次：
逐项核对：入口小屏算时间差，CC98 楼主编辑读编号，旧版规则统计证明类别。

第二次：
入馆记录看两次时间；23 是 CC98 回复楼层；证明数量来自旧版规则正文。

第三次及以后：
仍有字段与材料原文不一致。请按每张材料卡右侧标出的字段重新核对。

## 13. CC98：提交证据并完成 BD 四位口令

调查帖打开后，楼主编辑区即出现“上传证据”槽位；玩家可逐份提交已经取得的材料。

需要拖入四个道具：

1. 旧版离座规则。
2. 书包非本人证明。
3. 022 座位小票。
4. 本人来过证明。

全部上传后，帖子状态变为：

证据完整度：4/4
当前排名：04
提示：按证据顺序完成 BD 四位热度口令

进入互动前，系统必须解释 `bd = 帮顶`：对数字回复点击 `bd` 会把该数字按顺序写入口令。

口令顺序固定为上方四项上传材料从上到下：

1. 旧版离座规则：证明要求数量。
2. 书包非本人证明：身份通过项数量。
3. 022 座位小票：座位号末位。
4. 本人来过证明：到座耗时。

四项材料上传后新增 `24–31` 楼八条候选回复，显示数字 `4 / 3 / 1 / 0 / 5 / 2 / 6 / 7`。每条回复包含来源文本、数字和 `bd 选入` 按钮。

玩家按顺序选择四条回复，正确口令为：


## 18. 证据支线与最小可玩流程

座位凭据支线从首次到达 022 起开放。调查帖打开后，其余公示材料进入并行收集阶段：

1. 馆藏支线内部保持“终端解锁 → 题名检索 → 索书号 → 书架旧规则”。
2. 书包身份支线内部保持“现场拍照 → 调整亮度 → 生成报告 → 登记机盖章”。
3. 座位凭据支线可在首次到达 022 时用右向箭头推出夹缝小票，也可稍后补取。
4. 到馆证明支线的三项来源可按任意顺序收集；入馆记录、CC98 公示和旧版规则全部取得并读取后，才允许提交体艺补录。
5. 玩家取得任意一份材料后即可先上传 CC98，四份上传齐全后才开放 BD 四位口令。

入馆记录、找到 022、取得占座纸条和打开调查帖用于建立调查对象。座位小票可在首次到达 022 时提前取得；四项材料支线可按任意顺序交叉完成。

1. 操控 RPG 小人进入图书馆。
2. 到 022 发现占座书包。
3. 拿占座纸条。

5. 以任意顺序完成馆藏规则、书包身份、座位凭据和到馆证明四条支线；材料可逐份上传 CC98。

## 19. 开发验收标准

1. 玩家能从校园 RPG 进入图书馆，并在 022 触发占座书包剧情。
2. CC98 搜索占座纸条后出现 23 楼调查帖，且保留 bd。
3. 馆藏检索输入“三分钟离座法”能得到索书号 755。
4. 书架必须使用“索书号 755”才移开，不能直接点击拿规则。
5. 照片必须通过亮度调低才能识别书包标签。
6. 失物招领必须收到物品识别报告后才给出书包非本人证明；现场必须显示盖章机的进纸、扫描、机械下压、红章和出纸过程。
7. 022 座位小票必须用右向箭头推出；首次到达 022 即可完成，不受书包检查、调查帖或图书馆阶段限制，取得后保持当前阶段。
8. 体艺审核不再启动小人走路，只读取玩家已走过的 RPG 路径。
9. 体艺 7 / 47 / 3 三个答案有明确来源，不依赖隐喻。
10. CC98 必须先解释 `bd = 帮顶`；八条数字回复、四个口令槽、撤回、清空、错误提示和正确提交均可操作，正确口令 `3027` 使排名从 04 更新为 01。
11. 图书馆恢复申请必须在帖子到十大后才出现。
12. PASS 只能对 RPG 地图中的 022 书包生效。
13. 章节结尾必须解锁第三章任务“找到那本借走签到记录的书”。
14. 调查帖打开后，四条材料支线不能互相形成硬前置；任意已取得材料均可单独上传，终局仍要求四份齐全。
15. 第二章所有入口继续执行网络规则：浙大体艺仅允许移动数据，CC98 与浙大钉仅允许校园网；开发检查点必须预置对应网络，不能在剧情分支中豁免校验。
16. 第二章及后续章节新增道具时，手机与 RPG 道具栏都必须执行共享的像素飞入、接收脉冲和新槽位锁定动画；低动态偏好统一使用静态降级。
